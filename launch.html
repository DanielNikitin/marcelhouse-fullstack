Marcel House Website Launcher

Launch Options

Choose one of the following options to view the Marcel House website:

Recommended: Launch with PHP Server

For the best experience with full functionality including language switching:

Launch Website with PHP Server

This will start a local PHP server and open the website in your browser.

Alternative: Open HTML Directly

Open Website (HTML Only)

This will open the website directly from the file system. Some links and features may not work properly.

PHP Required

For a complete experience with language switching, PHP is required:

  1. Install PHP from windows.php.net
  2. Add PHP to your system PATH
  3. Run the batch file or click the "Launch Website with PHP Server" button above

Language Options

The website supports multiple languages:

Estonian (ET) English (EN) Finnish (FI) Russian (RU)

Note: Language switching requires the PHP server to be running.

About This Website

This is a static copy of the Marcel House WordPress website. It contains:

  • HTML files
  • CSS stylesheets
  • JavaScript files
  • Images and other media

The website is structured according to the original WordPress installation, but without the dynamic functionality that requires PHP and a database.
